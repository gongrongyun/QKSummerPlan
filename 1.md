# 前端计划
> ## 你画我猜
>> - 环境要求：安装[Node](https://nodejs.org/zh-cn/)环境(记得添加环境变量)
>> - 技术栈：JavaScript、npm、[webpack](https://webpack.js.org/)、[socket.io](https://socket.io/)
>> - 要求：
>>     1. 用户上线、离线通知
>>     2. 多人在线功能
>>     3. 绘画并广播至所有用户
>>     4. 发送消息并广播至所有用户
>>     5. 答案判定与积分计算
>>     6. 最终输赢判定
>> - 提示：JavaScript里的Canvas是个好东西，学习Js的时候不要略过哦
